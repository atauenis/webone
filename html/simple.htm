Welcome to %ServerName%!

This is a HTTP 1.x proxy server that makes old Internet browsers, media players and messengers usable again in the Web 2.0 world.

You need to set up your Web browser to use %ServerName% as HTTP Proxy Server. Also you may enter it in HTTPS and FTP Proxy Server lines. Or enter Proxy Automatic Configuration Script URL in your Web browser options:

http://%ServerName%/auto.pac

The current server is running WebOne %WOVer% on %WOSystem%.

Useful stuff

If you're seeing errors about "unknown certificate issuer" when opening sites over HTTPS, try to import proxy's Root Certificate. Note that to made HTTPS working you need at least Mozilla Firefox 1.0 or Opera 9 or Microsoft Internet Explorer 8.0 (also IE 6/7 is supported on systems like WinXP SP3, Vista). Also certificate import is required to use this proxy with MSN Messenger 4.7 and newer.

WebOne can be used to open FTP servers or to download videos from YouTube.
